更新内容：インベントリの実装 + ステータス計算機能の追加


1.インベントリの実装
新しく追加したFieldシーン内のTestオブジェクトに、テスト用のコンポーネントを追加しました。
Fieldシーン内で実行すると自動的にBattleシーンへ飛びますが、この際に、テスト用コンポーネント内の
knightsに設定したインベントリのデータがBattleシーン内の各味方ユニットに反映されます。

また、isSaveModeをTrueにして実行するとインベントリの内容が保存され、isLoadModeをTrueにして実行すると
前回にセーブしていたインベントリのデータが読み込まれます。

なお、敵のユニットに関してはインベントリが対応していないため、エディタで直接インベントリの値を変更する形になります。



2.ステータス計算機能の追加
インベントリで設定したユニットのデータベース,level,SPを基にして、Battleシーン内で各ステータスの値が計算されます。


計算式: 
BASE … 各ステータスの基礎値
TR … 各ステータスに固有の%値(0 ~ 1)
X = TR * BASE + (1-TR) * BASE * level/40 + SP
